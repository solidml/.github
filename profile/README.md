## What This Is

solidml is the definitive Solidity + ML reference — 131 repositories at the intersection of smart contract development and machine learning. This is the largest collection linking on-chain contract engineering with off-chain AI, covering everything from ML-optimized gas patterns to AI-generated Solidity to oracle-fed ML models.

**Solidity + ML:** Smart contract patterns, ML-optimized gas, AI contract generation, oracle integration, DeFi ML strategies, Solidity security with AI, automated testing, contract verification
